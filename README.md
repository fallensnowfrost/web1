# 企业员工与考勤管理系统

一个面向企业内部使用的员工与考勤管理系统静态网站原型，提供员工档案、考勤总览、班次规则、异常审批和快速打卡等核心管理界面。

## 功能亮点

- 总览仪表盘：展示在职员工、今日正常出勤、异常待处理和平均工时。
- 部门出勤概览：按部门查看出勤人数、总人数和出勤率。
- 员工档案：支持员工搜索、状态筛选和新增员工。
- 考勤记录：展示每日打卡、外勤、迟到和漏打卡等记录。
- 班次规则：维护标准班、弹性班、外勤班等规则。
- 审批管理：处理漏打卡、迟到说明、离职交接等审批事项。
- 交互体验：支持快速打卡、批量审批、深色模式和响应式布局。

## 快速开始

本项目是纯静态页面，不需要安装依赖或启动服务。

1. 克隆仓库：

   ```bash
   git clone https://github.com/fallensnowfrost/web1.git
   ```

2. 进入项目目录：

   ```bash
   cd web1
   ```

3. 直接用浏览器打开 `index.html`。

## 技术说明

- 使用原生 HTML、CSS 和 JavaScript 编写。
- 无第三方依赖，适合直接部署到 GitHub Pages、Nginx 或任意静态托管服务。
- 数据目前保存在前端脚本中，适合作为管理系统 UI 原型或二次开发基础。
